# 五一全民挖金矿活动原型 - 设计方案

## 选定方案：「黄金矿脉」国风游戏UI美学

**设计运动**：中国传统游戏大厅UI美学（Chinese Card Game Hall Aesthetic）

**核心原则**：
1. 高饱和度红金配色，还原截图中的游戏大厅氛围
2. 强烈的视觉层次：金色描边 + 深色背景 + 高亮CTA
3. 爽感动效：呼吸发光、金币飞溅、数字跳动
4. 新手引导：强路由 + 手指动画 + 高亮按钮

**色彩哲学**：
- 主色：深红 #8B0000 / 暗红 #6B0000
- 金色：#FFD700 / #FFA500
- 背景：深棕木纹 #2D1810
- 文字：金白 #FFF8DC

**布局范式**：
- 以截图为底图，在其上叠加UI层
- 右侧菜单栏保持原有位置，新增五一金矿入口
- 弹窗采用横屏全宽设计，符合游戏大厅风格

**签名元素**：
- 金色描边卡片
- 红色渐变按钮
- 金铲子SVG图标

**动效规范**：
- 呼吸发光：2s循环，scale 1.0→1.15，opacity 0.6→1.0
- 金币飞溅：从结算区飞向计数器，抛物线轨迹
- 数字跳动：0→目标值，0.8s弹性缓动
- Toast：从底部滑入，2s后淡出

**字体系统**：
- 标题：system-ui bold（模拟游戏字体）
- 正文：system-ui regular
- 数字：monospace bold（金币数量）
